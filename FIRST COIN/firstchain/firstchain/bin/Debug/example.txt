instructions pour tester le programme :
__________________________________________________________________________________________
__________________________________________________________________________________________
ETAPE A : créer des wallet
__________________________________________________________________________________________
[1] créer un portefeuille numérique en tapant createwallet
[2] créer un dossier "compte A" à la racine du programme
[3] déplacer les fichiers privatekey et publickey dans ce dossier
[4] faire la meme chose, mais cette fois, placez les clés dans un dossier à la racine appelé "compte B"

__________________________________________________________________________________________
ETAPE B : Miner mes premiers blocs pour gagner mes premiers coins
__________________________________________________________________________________________
En l'état, vous n'êtes toujours pas inscrit dans la chaine de blocs. Pour etre inscrit, vous devez soit miné des blocs jusqu'à
ce que ceux-ci soit ajouté à la chaine officiel OU faire une transaction avec quelquun qui vous donne des coins.
Nous allons nous enregistrer dans la chaine de bloc en minant. Nous n'avons pas encore de pointeur UTXO donc nous allons miner
en fantôme :
[1] miner les 80 premiers blocs avec le compte A, taper :
mine pkey:"location\compte A\publicKey" utxop:0 minlock:10000 ntime:80

plus en détail ->
pkey:# : { doit passer le chemin d'acces de la clé publique que vous souhaitez utiliser pour miner! }
utxop:#  { doit passer un pointeur utxo de votre clé publique inscrit dans le set d'utxo. 0 si vous n'en avez pas encore ! }
minlock:# { doit passer une valeure en seconde définissant quel locktime maximum ont les transactions que vos blocs auront pendant le minage :
            par exemple en mettant 10000, les transactions pourissant dans 10000 secondes ne seront pas intégrés ! }
ntime:#  { doit passer une valeure definissant le nombre de blocs miné souhaité avant la fin du processus de minage! si 0, c'est à l'infini! }

[2] Vous devez maintenant avoir un pointeur UTXO. Pour le connaître, taper :
getutxop pkey:"location\compte A\publicKey"
Votre utxo pointeur est 4. Gardez le en mémoire! Votre solde est de 50. Mais vos autres pointeur sont aussi 44, 84, 124... 
puisque vous en avez miné à blanc 80 blocs...
[2] Miner a nouveau cette fois ci en utilisant votre pointeur.
mine pkey:"location\compte A\publicKey" utxop:4 minlock:10000 ntime:10
[3] Cette fois, vous avez miné en utilisant votre wallet ! Votre solde a été mis à jour! Pour le vérifier, tapez : 
getutxo 4
Puisque vous avez miné 10 nouveau bloc, vous avez gagnez 50 coin par bloc, soit 500 coin. Vous avez maintenant un solde de 550 !

__________________________________________________________________________________________
ETAPE C : Créer une transaction et l'ajouter à la chaine de bloc
__________________________________________________________________________________________

[1] vous allez donnez avec le compte A 10 coin au compte B ! Pour cela vous devez d'abord créer une transaction. Taper : 
newtx sprkey:"location\compte A\privateKey" spukey:"location\compte A\publicKey" sutxop:4 amount:10 rpukey:"location\compte B\publicKey" rutxop:0 fee:20 lock:1699981949

plus en détail ->
sprkey:# : { doit passer le chemin d'acces de la clé privé que vous souhaitez utiliser ! }
spukey:#   { doit passer le chemin d'acces de la clé publique que vous souhaitez utiliser ! }
sutxop:#   { doit passer le pointeur de votre clé publique dans le set d'utxo!  }
amount:#   { doit passer le nombre de coin que vous donnez au receveur! }
rpukey:#   { doit passer le chemin d'acces de la clé publique du receveur!  }
rutxop:#   { doit passer le pointeur de la clé publique du receveur dans le set d'utxo! Peut être 0. }
fee:#      { doit passer les frais de transactions qui seront appliqués. Ils doivent recquiert le minimum frais de transaction de la blockchain actuel! Les extras, 
            c'est pour le mineur ! }
lock:#     { doit passer le temps en seconde (empreinte temporelle UNIX!) avant laquel vous désirez que cette transaction soit inscrite dans la blockchain officielle !
             votre transaction ne sera plus valable à partir du moment où ce temps sera passé! Prévoyez un délai plus ou moins long en fonction de l'activité du réseau !  }

[2] Intégrer cette transaction à vos transaction en attente afin de l'intégrer aux prochains blocs que vous allez miner. Taper : 
reqtx "location\TX11"
[3] Maintenant miner 10 autres blocs avec votre compte A ! 
mine pkey:"location\compte A\publicKey" utxop:4 minlock:10000 ntime:10
[4] Pour vérifier que vous avez bien intégrer Compte B à la chaine de blocs et que vous lui avez donner 10 coin taper : 
getblockinfo 91
Vous devriez voir votre transaction inscrite dans ce bloc! 
[5] Vous pouvez maintenant utiliser ce compte B pour miner et effectuer des transaction ! Taper : 
getutxop pkey:"location\compte B\publicKey"
Vous obtenez pour le compte B un pointeur égal a 3204 et un solde de 10 coin! 
La prochaine fois, si vous voulez augmentez le solde de coin du compte B miner avec celui-ci : 
mine pkey:"location\compte B\publicKey" utxop:3204 minlock:10000 ntime:10
